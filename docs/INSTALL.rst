Sinar.Theme Installation
------------------------

To install Sinar.Theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``Sinar.Theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        Sinar.Theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
